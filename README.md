# Portfolio Website - Kishor Gogoi

A responsive portfolio website built with **Vite + React + Tailwind CSS**.

## Prerequisites
- Node.js installed on your system.

## Setup Instructions

1.  **Open Terminal** in this folder (`d:\Project\Professionalp`).
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Start Development Server**:
    ```bash
    npm run dev
    ```
4.  **Open in Browser**:
    Click the link shown in the terminal (usually `http://localhost:5173`).

## Build for Production
To create a production build:
```bash
npm run build
```
The output will be in the `dist` folder.
